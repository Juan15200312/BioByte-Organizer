# BioByte-Organizer

SmartFile Organizer es un asistente automatizado que actúa como un "sereno" para tu carpeta de descargas. Su función
principal es vigilar la carpeta y, de forma automática o bajo demanda, clasificar cada archivo en su lugar
correspondiente (carpetas de Imágenes, Documentos, Vídeos, etc.). 

## Instalar y Activar .venv

## Windows
- python -m venv .venv
- .venv\Scripts\activate


## Linux
- python3 -m venv .venv
- source .venv/bin/activate


## Generar e instalar librerias
- pip freeze > requirements.txt
- pip install -r requirements.txt
